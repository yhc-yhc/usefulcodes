searchPhotosByImage

Method : POST 
desc : 传入一张带有人脸的图片，在数据库中检索，返回共享照片中含有此人的图片

name required type des default
file yes Binary 上传文件的二进制数据 -
siteId yes String 乐园的siteId -
date yes String 时间的字符串格式，YYYY/MM/DD -
